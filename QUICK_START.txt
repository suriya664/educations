    METHOD 1: EASY WAY (Using Batch Files)

Step 1: Run git-setup.bat
   - Double-click: git-setup.bat
   - This will initialize Git and create first commit
   
Step 2: Create GitHub Repository
   - Go to: https://github.com/new
   - Name: learnsmart-academy
   - Make it PUBLIC
   - DO NOT add README or .gitignore (we have them!)
   - Click "Create repository"
   
Step 3: Run git-push.bat
   - Double-click: git-push.bat
   - Enter your GitHub username when asked
   - Files will upload to GitHub!
   
Step 4: Enable GitHub Pages
   - Go to your repository on GitHub
   - Click Settings → Pages
   - Source: main branch
   - Click Save
   - Wait 2 minutes
   - Your site is LIVE! 🚀

    METHOD 2: MANUAL WAY (Command Line)
================================================================================

Step 1: Open Command Prompt or PowerShell
   - Press Win + R
   - Type: cmd
   - Press Enter

Step 2: Navigate to Project
   cd c:\smartfusion\EDUCATIONONLIN

Step 3: Initialize Git
   git init
   git add .
   git commit -m "Initial commit: LearnSmart Academy"

Step 4: Create GitHub Repository
   - Go to: https://github.com/new
   - Name: learnsmart-academy
   - Make it PUBLIC
   - Create repository

Step 5: Push to GitHub (Replace YOUR_USERNAME)
   git remote add origin https://github.com/YOUR_USERNAME/learnsmart-academy.git
   git branch -M main
   git push -u origin main

Step 6: Enable GitHub Pages
   - Settings → Pages → Source: main → Save
